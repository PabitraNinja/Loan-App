# Personal Loan & Interest Tracking App - UI/UX Prototype

This is a modern, professional fintech UI/UX design for a Personal Loan & Interest Tracking application, optimized for Android mobile screens.

## Design Highlights
- **Framework:** React 18 + TypeScript
- **Styling:** Vanilla CSS following Material Design 3 guidelines.
- **Color Palette:** Navy Blue (`#1A237E`) for professionalism, Success Green (`#4CAF50`) for profit, and Light Grey (`#F5F7FA`) for clean layouts.
- **Icons:** Lucide-React for clear visual metaphors.
- **Mobile First:** Designed for a 360-500px width experience.

## Screens Included
1. **Dashboard:** High-level financial overview and recent activity.
2. **Loans List:** Searchable and filterable borrower management.
3. **Loan Details:** Deep dive into specific loan terms and payment history.
4. **Add Loan:** Clean, structured form for onboarding new borrowers.
5. **Add Payment:** Quick entry for interest or principal repayments.
6. **Reports:** Visualized profit and distribution data.
7. **Reminders:** Due date tracking with one-tap notification triggers.
8. **Settings:** Application configuration and profile management.

## How to Run
1. Ensure Node.js is installed.
2. Run `npm install`.
3. Run `npm start`.
